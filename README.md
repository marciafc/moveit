# Moveit

Instalar yarn

    npm install --global yarn

    yarn --version

## Criar projeto com typescript

```shell
	yarn create react-app moveit --template=typescript

	OU

	npm create react-app moveit --template=typescript
```

## Start na aplicação

```shell

	yarn start

	http://localhost:3000

```
